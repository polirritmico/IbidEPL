[size=150][b][color=#008000]Descripción:[/color][/b][/size]
Una pequeña herramienta para manejar las notas ibíd.

[b][color=#BF8000]Requisitos[/color]:[/b] Sigil >0.9.8
[b][color=#BF8000]Instalación[/color]:[/b] Igual a cualquier plugin de Sigil, van al menú: Preferencias → Complementos → Añadir complemento. Allí seleccionan el zip con el plugin y pulsan Abrir. Opcionalmente pueden agregarlo a la barra de accesos rápidos.
[b][color=#BF8000]Ejecución:[/color][/b] Primero abrir el xhtml con las notas o seleccionarlo en el Explorador del libro, a continuación abrir el plugin. Para que funcione, el archivo de notas debe seguir el estándar EPL. Se muestran recuadros de ayuda dejando el ratón sobre la interfaz sin pulsar clic.
[b][color=#BF8000]Link de descarga:[/color][/b] [url=https://mega.nz]versión 1.0[/url]

[b][color=#BF8000]Atajos:[/color][/b]
Ctrl + Rueda del ratón sobre el ibid o la nota para agrandar o reducir el tamaño del texto.
Ctrl + AvPág: Abre nota ibid siguiente
Ctrl + RePág: Abre nota ibid anterior
Ctrl + Retroceso (Backspace): Procesar Ibid
Ctrl + Enter: Modificar nota ibid

[size=150][b][color=#008000]Funcionamiento:[/color][/b][/size]
Por defecto el plugin agrega la etiqueta «<i xml:lang="la">Ibid</i>» y el Separador «TEXTO_ADICIONAL:» (Ambas se pueden ajustar en la ventana de Opciones). El separador pretende ser solo una ayuda visual para diferenciar la nota original del ibid (e incluso para hacer búsquedas desde Sigil).
La idea es procesar la nota y luego hacer el ajuste manual en el mismo recuadro del ibid, que nos deja modificar el texto como queramos. El borde cambiará de color recordándonos que lo hemos editado pero que todavía no hemos guardado esas modificaciones. Cuando presionemos el botón Modificar estaremos realmente modificando la nota y luego cuando le demos a Aceptar se escribirán todos las modificaciones en el xhtml.

Si son muchas notas se puede utilizar el botón «Procesar y modificar todo». Se sugiere utilizar los atajos de teclado disponibles para las tareas más recurrentes.

[size=150][b][color=#008000]Ejemplo de uso:[/color][/b][/size]
[list=1][*]Vamos a la primera nota ibíd. y editamos el texto.
[*]Pulsamos Ctrl + Enter para modificar la nota y presionamos Ctrl + AvPág para ir a la siguiente nota íbid.
[*]Pulsamos Ctrl + Retroceso para procesar la nota y ajustamos manualmente.
[*]Repetimos 2 y 3 hasta terminar con todas las notas ibíd.
[*]Pulsamos Aceptar para confirmar los cambios y sobrescribir el archivo de notas.[/list]
