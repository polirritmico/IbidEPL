Titivillus

------------------

El plugin abre el archivo seleccionado.
Recuadros de ayuda dejando el ratón sobre la interfaz sin pulsar clic.

Ctrl + Rueda del ratón sobre el ibid o la nota: Agranda o reduce el tamaño del texto.
Ctrl + AvPág: Abrir nota ibid siguiente
Ctrl + RePág: Abrir nota ibid anterior
Ctrl + Backspace: Procesar Ibid
Ctrl + Enter: Modificar nota ibid
